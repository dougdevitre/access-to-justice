## Why two review bodies

The Methodology Board decides what to measure and how. The Data Ethics Review decides whether measuring would cause harm. Splitting these functions prevents a compelling methodological idea from overwhelming ethical concerns at the same table.

## Why governance is slow on purpose

The adoption workflow takes 90+ days from proposal to publication: proposal → Data Ethics Review → 60-day public comment → Board vote → 90-day implementation lag. This pace matches the pilot's quarterly reporting cycle rather than social-media time. Transparency without governance is a harm vector; the slow pace is a feature.
