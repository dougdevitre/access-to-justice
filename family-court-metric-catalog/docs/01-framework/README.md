## Reading order for newcomers

1. Start with document 00 to understand the proposition.
2. Read document 01 for the framework.
3. Skim document 02 to see the shape of the catalog.
4. Reference document 19 when you need to look up a specific metric.

Document 03 (Disclosure Tiers) is a reference document — read it when a tier assignment surfaces in another document.

Document 20 (Binder Index) is the navigation document for the full binder; use it to find audience-specific reading paths into documents in the other directories.
